# Domain-Scanner
Commands Summary:

If on Ubuntu:
Use a virtual environment:
	python3 -m venv myenv
	source myenv/bin/activate

Install libraries:

	pip install whois
	pip install dnspython
	pip install requests
	pip install pyopenssl

Run the tool:

	python3 proj.py
